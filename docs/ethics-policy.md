# Ethics and Usage Policy

## Purpose
This project is a **research prototype** for indoor localization using multi-sensor fusion. It is not intended for covert surveillance or any application that violates privacy rights.

## Prohibited uses
- Tracking or monitoring individuals without explicit, informed consent.
- Use in locations where participants are not clearly notified and able to opt out.
- Deployment for law enforcement or security without appropriate legal authorization.
- Any activity that violates local, regional, or national privacy regulations.

## Required safeguards
- **Consent**: All participants must opt in with informed consent.
- **Data minimization**: Store only what is necessary; delete data promptly.
- **Access control**: Restrict access to data, logs, and outputs.
- **Auditability**: Maintain logs of access and changes.

## Acknowledgment
By using or contributing to this repository, you agree to follow this policy and all applicable laws.
